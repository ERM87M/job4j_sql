Репозиторий для практических заданий по курсу SQL в рамках обучения на платформе job4j.
Описание
В данном проекте реализована схема базы данных интернет-магазина на PostgreSQL
схема интернет-магазина из 4 таблиц(users
products
orders
order_items)
Между таблицами настроены связи с помощью внешних ключей. Также используются ограничения PRIMARY KEY, NOT NULL, UNIQUE, CHECK и значения по умолчанию. Схема позволяет хранить информацию о пользователях, товарах, заказах и составе каждого заказа.
